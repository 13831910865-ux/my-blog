Markdown 是一种轻量级的标记语言，让你可以用简单的语法写出格式丰富的文章。

## 基本语法

### 标题

使用 `#` 号表示标题层级：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调

- **加粗**：`**文字**`
- *斜体*：`*文字*`
- ~~删除线~~：`~~文字~~`

### 列表

无序列表：

- 项目一
- 项目二
- 项目三

有序列表：

1. 第一步
2. 第二步
3. 第三步

### 代码

行内代码使用反引号：`console.log('hello')`

代码块使用三个反引号：

```javascript
function greet(name) {
  return `你好，${name}！`;
}
```

### 链接和图片

- 链接：`[文字](URL)`
- 图片：`![描述](图片URL)`

### 引用

> 这是一段引用文字

---

以上就是最常用的 Markdown 语法，开始写作吧！
